Welcome to Ollama Demo Application

About This Application

This is a simple Spring Boot application that demonstrates how to integrate with Ollama, an open-source platform for running large language models locally.

The application provides REST API endpoints to interact with Ollama models and generate text responses.

Try It Out

You can test the Ollama integration using our interactive test page:

Go to Test Page

Available API Endpoints:

  • POST /api/simple - Simple API using Java's HttpClient
  • POST /api/direct - Direct API using HttpURLConnection

Both endpoints accept a JSON payload with a message field and return a response from the Ollama model.

Example request: 

{
  "message": "Tell me a joke"
}
